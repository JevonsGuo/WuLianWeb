# 产品展示页 - 三栏布局改造需求文档

## 一、需求概述

将现有产品展示页改造为**三栏联动布局**，提供类似文件管理器或产品目录的浏览体验。用户从左到右逐步筛选，最终查看产品详情。

### 核心交互流程
选择产品大类 → 展示该类产品列表 → 选择具体产品 → 展示产品详情（带标签页）

text

## 二、布局结构（桌面优先）

### 整体布局（三栏等宽或自适应）

| 左侧栏（第一列） | 中间栏（第二列） | 右侧栏（第三列） |
|---------|---------|---------|
| 宽度：280px | 宽度：320px | 剩余宽度（自适应） |
| 产品大类列表 | 产品列表 | 产品详情（标签页） |

**响应式行为（可选，第一版可不实现）**：
- 平板：三栏可横向滚动
- 手机：单栏显示，点击进入下一级（类似导航栈）

## 三、左侧栏：产品大类列表

### 功能要求
- 垂直列出所有产品大类
- 每个大类显示：
  - 缩略图（40x40px，圆角）
  - 大类名称
- 支持**选中状态高亮**（背景色变化 + 左侧边框指示条）
- 点击大类时：
  - 更新中间栏的产品列表
  - 清空空/重置右侧栏的产品详情

### 数据结构

八、数据库迁移 SQL

sql
-- 1. 为 products 表添加新字段
ALTER TABLE products 
ADD COLUMN main_image_url TEXT,
ADD COLUMN summary_content TEXT,
ADD COLUMN specifications_content TEXT;

-- 2. 创建 product_attachments 表
CREATE TABLE product_attachments (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  product_id UUID NOT NULL REFERENCES products(id) ON DELETE CASCADE,
  file_name TEXT NOT NULL,
  file_url TEXT NOT NULL,
  file_type TEXT NOT NULL,
  file_size INTEGER,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 3. 创建 Storage Bucket（如果还没有）
-- 在 Dashboard 手动创建：product-attachments

-- 4. 关闭 RLS（开发环境）或配置策略
ALTER TABLE product_attachments DISABLE ROW LEVEL SECURITY;
